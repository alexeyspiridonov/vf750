# Carburetor cleaning

- Disassemble all four carbs and note jet sizes and positions before mixing parts.
- Ultrasonic cleaning is the best method for VF750 carb bodies.
- Clear all jets, passages, and pilot circuits with compressed air.
- Replace float valve needles if they are stuck or worn.
- If a bowl stays dry even though fuel reaches the rack, inspect the float valve, seat filter, float height, bowl vent, and inlet rail before changing jet sizes.
- Reference video: [Honda VF750 Magna Carb Clean - TheMotorcycleMD](https://www.youtube.com/watch?v=52k8LaD3q4Y)
