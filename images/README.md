# Image References

This folder collects visual references for the `1982 Honda VF750S Sabre` project.

## Folders

- `stock/` — stock Sabre reference photos
- `cafe-racer/` — already collected finished custom-bike photos
- `pinterest/` — Pinterest visual inspiration
- `build-process/` — internet-sourced build and conversion process photos
- `repair-service/` — internet-sourced service and repair reference thumbnails
