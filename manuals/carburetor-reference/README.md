## Scope Notes

- The project bike context is `1982 Honda VF750S Sabre`.
- The factory shop manual here is directly useful because it covers the `1982 V45 Sabre/VF750S`.
- The Haynes material is broader and includes multiple Sabre/Magna years.
- The local fiche is for the `1984-1985 VF700S Sabre`, so use it for component families, routing ideas, and part-family comparison, but verify the exact 1982 part number before ordering.

## Practical Order Of Use

1. Start with `manuals/04-shop-manual-fuel-system.pdf` for the factory carburetor procedures.
2. Use `manuals/03-shop-manual-maintenance.pdf` for synchronization, idle, choke, air cleaner, and routine carb-adjacent service.
3. Use `manuals/02-haynes-fuel-and-exhaust-systems.pdf` when you want a second explanation or a broader teardown sequence.
4. Use `parts-catalog/01-carburetor-assy-and-components.pdf` and `parts-catalog/04-air-cleaner.pdf` when identifying assemblies, hoses, and missing hardware.
5. Use the notes in `notes/` for the fastest shortlist of recurring failures, quick procedures, and existing video references already collected in the repo.
