# Проверка системы зарядки

1. Проверьте напряжение аккумулятора при выключенном двигателе — около 12,6 В.
2. Запустите двигатель и увеличьте обороты до 3000.
3. Напряжение зарядки должно быть 13,5–14,8 В.
4. Если напряжение низкое, проверьте выходное напряжение статора на его выводах.
5. Ожидается 60–80 В переменного тока при 5000 об/мин.
6. Если статор исправен, а зарядка низкая — вероятно, неисправен регулятор/выпрямитель.
