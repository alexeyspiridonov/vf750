## YouTube-референсы

### Полезные видео по VF750 / V45 Sabre / Magna

- [Honda VF750 Magna Carb Clean - TheMotorcycleMD](https://www.youtube.com/watch?v=52k8LaD3q4Y)
  Подробный разбор и чистка карбюраторов. Самый практичный ролик по карбюраторам из собранных здесь.

- [Syncing The Carburetors On The Honda VF750 - Revive 2 Ride](https://www.youtube.com/watch?v=EXc9i7zoA9g)
  Хороший реальный пример синхронизации карбюраторов на VF750.

- [HONDA V45 MAGNA / VF750C / RC09 1982 Carb sync / reglage - ichikonti](https://www.youtube.com/watch?v=3buvmQb6Ogs)
  Короткий, но полезный ролик именно по раннему VF750C / V45 Magna.

- [Honda V45 Magna Carb Sync Tool - HighTechLab](https://www.youtube.com/watch?v=D0zCyCsNFt4)
  Референс по самодельному инструменту для синхронизации.

- [1982 Magna V45 Final work: Carbs install, valve clearance and Synch! - Moto-Resto Machine and Repair](https://www.youtube.com/watch?v=BsNvGBBtUAc)
  Полезный комбинированный референс по установке карбюраторов, клапанам и финальной синхронизации.

- [Honda VFR750 vs VF750 valve adjustment wisdom and a visit from the DUMB VIDEO POLICE! Watch and see! - Moto-Resto Machine and Repair](https://www.youtube.com/watch?v=dRjBuBrFGhM)
  Полезнее как разговор о клапанном механизме, распредвалах и маслоподаче, чем как пошаговая инструкция.

- [Honda Magna V45 Cafe Racer Style - ICCustomsUSA](https://www.youtube.com/watch?v=UlwVQwkROtU)
  Хороший визуальный референс по cafe racer на базе семейства Magna/Sabre.

- [NEW 1983 HONDA MAGNA V45 CAFE RACER PROJECT - AvidTV](https://www.youtube.com/watch?v=9gkjgG-AY7E)
  Короткий обзор проекта, полезный в основном как визуальное направление.

- [Honda VF750 Cafe Racer by Lucky Customs - NakedBikesNation](https://www.youtube.com/watch?v=qPZOBHhdTv8)
  Скорее вдохновение, чем техническая документация, но для визуального сравнения полезно.

### Краткий вывод по YouTube

- Лучше всего покрыты темы карбюраторов, синхронизации, базового обслуживания и визуальных кастомов.
- Слабо покрыта тема полного EFI / ECU swap именно для VF750.
- Для исследования EFI-конверсии форумы пока полезнее YouTube.
