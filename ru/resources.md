## YouTube

### Полезные видео по VF750 / V45 Sabre / Magna

- [Honda VF750 Magna Carb Clean - TheMotorcycleMD](https://www.youtube.com/watch?v=52k8LaD3q4Y)  
  Подробная разборка и чистка карбюраторов.

- [Syncing The Carburetors On The Honda VF750 - Revive 2 Ride](https://www.youtube.com/watch?v=EXc9i7zoA9g)  
  Практический пример синхронизации карбюраторов на VF750.

- [HONDA V45 MAGNA / VF750C / RC09 1982 Carb sync - ichikonti](https://www.youtube.com/watch?v=3buvmQb6Ogs)  
  Короткое видео именно по раннему VF750C / V45 Magna.

- [Honda V45 Magna Carb Sync Tool - HighTechLab](https://www.youtube.com/watch?v=D0zCyCsNFt4)  
  Референс по самодельному инструменту для синхронизации.

- [1982 Magna V45 Final work - Moto-Resto Machine and Repair](https://www.youtube.com/watch?v=BsNvGBBtUAc)  
  Установка карбов, регулировка клапанов и финальная синхронизация.

- [Honda VFR750 vs VF750 valve adjustment wisdom - Moto-Resto Machine and Repair](https://www.youtube.com/watch?v=dRjBuBrFGhM)  
  Больше полезно как обсуждение клапанного механизма, распредвалов и смазки.

- [Honda Magna V45 Cafe Racer Style - ICCustomsUSA](https://www.youtube.com/watch?v=UlwVQwkROtU)  
  Визуальный cafe-racer референс по семейству Magna/Sabre.

- [NEW 1983 HONDA MAGNA V45 CAFE RACER PROJECT - AvidTV](https://www.youtube.com/watch?v=9gkjgG-AY7E)  
  Короткий обзор проекта, больше как визуальная идея.

- [Honda VF750 Cafe Racer by Lucky Customs - NakedBikesNation](https://www.youtube.com/watch?v=qPZOBHhdTv8)  
  Скорее вдохновение, чем техническая документация.

### Вывод по YouTube

- Лучше всего покрыты карбы, синхронизация, базовое обслуживание и визуальные кастомы.
- Слабее всего покрыта полная EFI / ECU конверсия именно VF750.
- Для EFI сейчас полезнее форумы, чем YouTube.
