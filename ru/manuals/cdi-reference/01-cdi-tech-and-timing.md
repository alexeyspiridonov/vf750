# Техника CDI / Spark Unit и углы опережения

## 1. Что реально стоит на мотоцикле

Для раннего V45 Sabre локальные мануалы описывают battery-fed transistorized ignition system со следующим набором узлов:

- две pulse generator coils;
- два spark units;
- две ignition HT coils;
- wasted-spark схема.

Два spark units обслуживают пары цилиндров:

- цилиндры `1 и 3`;
- цилиндры `2 и 4`.

На ранних Sabre 700/750 spark units установлены в задней части мотоцикла под сиденьем / в районе заднего крыла.

## 2. Штатные данные по углу опережения

Для `1982-1983 750 Sabre` локальный комплект мануалов даёт такие практические контрольные точки:

| Параметр | Значение |
| --- | --- |
| Опережение на холостом ходу | `10° BTDC` |
| Начало роста опережения | около `1,500 rpm` |
| Полное опережение | `37° BTDC` при `3,300 rpm` |

Эти числа совпадают между локальным OCR заводского раздела по обслуживанию и локальным Haynes-разделом по зажиганию.

## 3. Что известно о штатной кривой опережения

Заводской мануал и Haynes дают контрольные точки проверки, но не полную внутреннюю карту spark units по всем оборотам.

Практическая интерпретация:

- на холостом мотоцикл находится примерно на `10° BTDC`;
- примерно до `1,500 rpm` угол остаётся близок к базовому;
- затем spark units электронно наращивают опережение;
- к `3,300 rpm` система должна выйти на `37° BTDC`.

Это означает, что сервисная литература даёт надёжную `точку начала` и `точку полного опережения`, но не скрытую OEM-кривую по всем промежуточным точкам.

## 4. Важные спецификации для диагностики

Релевантные локальные данные:

| Параметр | Значение |
| --- | --- |
| Номинальное сопротивление pulse generator | `480 ohms +/- 10%` |
| Диапазон проверки Haynes для 1982-1986 | `450 to 550 ohms` |
| Тип системы | `12 V transistorized ignition` |
| Регулировка угла | `none` |

Релевантные пары проводов pulse generators в локальном Haynes OCR:

- `white/yellow` и `yellow` для одной пары pickup;
- `white/blue` и `blue` для другой пары.

Поскольку OCR неидеален, всегда сверяйте цвета проводов с локальной wiring diagram, прежде чем depin-ить или repin-ить что-либо.

## 5. Что в стоке не регулируется

Локальные мануалы однозначны по сути:

- штатной процедуры регулировки зажигания нет;
- система проверяется timing light, а не регулируется;
- если угол неправильный, значит неисправен компонент, и его нужно проверить или заменить.

Это важно, потому что многие проблемы зажигания на старых мотоциклах ошибочно объясняют `уплывшим зажиганием`. На этой платформе проблема обычно не в регулировке, а в одном из следующих пунктов:

- плохой spark unit;
- слабый сигнал от pulse generator;
- плохое питание по напряжению;
- плохая масса;
- проблемы катушки или разъёмов.

## 6. Практическая сводка по штатной кривой

Если нужен один рабочий ориентир для исправного штатного VF750S, он такой:

- здоровое питание 12 В;
- оба spark units чисто отрабатывают свой ряд цилиндров;
- базовый угол около `10° BTDC`;
- чистый рост опережения до `37° BTDC` к `3,300 rpm`.

Всё, что выходит за эти рамки, сначала следует считать неисправностью, а не возможностью для тюнинга.
