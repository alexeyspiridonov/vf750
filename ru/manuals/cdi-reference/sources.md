## Пояснения по интерпретации

- Локальные материалы Honda / Haynes используются как первичный источник по стоковым спецификациям и контрольным точкам угла опережения.
- Ignitech используется как основной источник по различию современных TCI и CDI продуктов.
- Carmo используется как основной источник по текущему наличию готовых replacement modules.
- Форумные материалы полезны для шаблонов симптомов, но не считаются главным источником совместимости железа.
