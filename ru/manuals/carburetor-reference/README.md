# Пакет справочных материалов по карбюраторам

Эта папка собирает карбюраторные материалы, доступные в репозитории для проекта Honda Sabre.

## Что включено

### Выжимки из мануалов
- `manuals/01-haynes-tune-up-and-routine-maintenance.pdf`
- `manuals/02-haynes-fuel-and-exhaust-systems.pdf`
- `manuals/03-shop-manual-maintenance.pdf`
- `manuals/04-shop-manual-fuel-system.pdf`
- `manuals/05-shop-manual-1983-addendum-carb-updates.pdf`

### Выжимки из каталога деталей
- `parts-catalog/01-carburetor-assy-and-components.pdf`
- `parts-catalog/02-throttle-and-choke-cables.pdf`
- `parts-catalog/03-fuel-tank-and-fuel-valve.pdf`
- `parts-catalog/04-air-cleaner.pdf`

### Заметки
- `notes/howto-carburetors.md`
- `notes/common-issues.md`
- `notes/videos-and-links.md`
- `manifest.json`

## Ограничения по охвату

- Контекст проектного мотоцикла: `1982 Honda VF750S Sabre`.
- Локальный factory shop manual полезен напрямую, потому что он покрывает именно `1982 V45 Sabre / VF750S`.
- Материалы Haynes шире и включают несколько годов Sabre/Magna.
- Локальный fiche в репозитории относится к `1984-1985 VF700S Sabre`, поэтому используйте его для понимания семейств деталей, маршрутов и сравнения узлов, но конкретный номер детали для `1982` обязательно перепроверяйте перед заказом.

## Практический порядок использования

1. Начинайте с `manuals/04-shop-manual-fuel-system.pdf` для заводских процедур по карбюраторам.
2. Используйте `manuals/03-shop-manual-maintenance.pdf` для синхронизации, холостого хода, подсоса, air cleaner и соседних сервисных операций.
3. Открывайте `manuals/02-haynes-fuel-and-exhaust-systems.pdf`, когда нужен второй взгляд на процедуру или более общий порядок разборки.
4. Используйте `parts-catalog/01-carburetor-assy-and-components.pdf` и `parts-catalog/04-air-cleaner.pdf`, когда нужно идентифицировать узлы, шланги и недостающий крепёж.
5. Держите заметки из `notes/` под рукой как самый быстрый shortlist по повторяющимся проблемам, быстрым процедурам и уже собранным видео-ссылкам.
