# Видео и ссылки

Это карбюраторные референсы, уже собранные в репозитории и полезные рядом с выжимками из мануалов.

## Видео, уже присутствующие в наборе

- Honda VF750 Magna Carb Clean - TheMotorcycleMD

  Подробный референс по разборке и чистке.

- Syncing The Carburetors On The Honda VF750 - Revive 2 Ride  
  https://www.youtube.com/watch?v=EXc9i7zoA9g

- HONDA V45 MAGNA / VF750C / RC09 1982 Carb sync / reglage - ichikonti  
  https://www.youtube.com/watch?v=3buvmQb6Ogs

- Honda V45 Magna Carb Sync Tool - HighTechLab  
  https://www.youtube.com/watch?v=D0zCyCsNFt4  
  Референс по самодельному инструменту для синхронизации.

- 1982 Magna V45 Final work: Carbs install, valve clearance and Synch! - Moto-Resto Machine and Repair  
  https://www.youtube.com/watch?v=BsNvGBBtUAc
