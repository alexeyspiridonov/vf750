# Интернет-источники

Проверено: 2026-04-12

## Wiring diagrams и OEM-каталоги

- Local electrical-reference package  
  Сначала используйте локальные копии в репозитории:
  - `manuals/03-wiring-diagrams-all-models.pdf`
  - `manuals/04-wiring-diagram-1982-750-sabre.pdf`

- Историческая заметка: старый индекс wiring diagrams на V4MuscleBike по состоянию на 2026-04-12 выглядит недоступным, поэтому больше не считается primary source.

- Partzilla 1982 VF750S V45 Sabre OEM parts root  
  https://www.partzilla.com/catalog/honda/motorcycle/1982/vf750s-a-v45-sabre  
  Полезно, когда локальный каталог деталей VF700S 84/85 не совпадает с проектным мотоциклом 1982 года.

- Partzilla 1984 VF700S Sabre OEM parts root  
  https://www.partzilla.com/catalog/honda/motorcycle/1984/vf700s-a-sabre  
  Полезный вспомогательный reference для локального fiche VF700S.

## Зажигание / spark / CDI / pulse generator

- 82 saber v45 spark  
  https://v4musclebike.com/threads/82-saber-v45-spark.47509/  
  Полезно для расположения spark units и логики поиска проблем по переднему/заднему ряду.

- 84 vf 750 sabre ignition problem  
  https://v4musclebike.com/threads/84-vf-750-sabre-ignition-problem.32922/  
  Полезно для no-spark диагностики, проверки сопротивления pulse generator, проверки питания и массы CDI, а также связи с тахометром заднего ряда.

- CDI units  
  https://v4musclebike.com/threads/cdi-units.5971/  
  Полезно как reference при оценке replacement spark box / CDI options.

## Charging system / stator / regulator-rectifier

- Charging question - is this normal, or a sign?  
  https://v4musclebike.com/threads/charging-question-is-this-normal-or-a-sign.39080/  
  Полезно для типовой последовательности: сначала разъёмы, затем проверки статора и regulator/rectifier.

- Melted Regulator/Rectifier Plug  
  https://v4musclebike.com/threads/melted-regulator-rectifier-plug.38335/  
  Хорошая сводка по очень распространённой точке отказа и по важности обслуживания разъёмов.

- 3 yellow wire fix  
  https://v4musclebike.com/threads/3-yellow-wire-fix.790/  
  Очень полезное community-описание проблемы разъёма статора и hardwire fix.

- Mosfet R/R mod  
  https://v4musclebike.com/threads/2876/  
  Полезный путь апгрейда, если штатный regulator/rectifier умер или зарядка нестабильна.

- I need a stator......Rick's or Electrosport?  
  https://v4musclebike.com/threads/i-need-a-stator-ricks-or-electrosport.37309/  
  Полезно по реальным симптомам отказа статора на Sabre и обсуждению replacement options.

- 1984 Sabre V45 750cc charging system problem  
  https://v4musclebike.com/threads/1984-sabre-v45-750cc-charging-system-problem.16712/  
  Полезно тем, что связывает проблемы зарядки, spark boxes, поведение тахометра и типовые V4-неисправности.

## Starter relay / main fuse / harness

- Solenoid replacement, MAIN FUSE FIX!  
  https://v4musclebike.com/threads/solenoid-replacement-main-fuse-fix.3223/  
  Очень полезный reference по замене старого solenoid с dogbone fuse и по популярной blade-fuse conversion.

- Starter Relay and Fuse  
  https://v4musclebike.com/threads/starter-relay-and-fuse.47255/  
  Полезно по современным вариантам замены и теме dogbone-fuse replacement.

- starter relay question  
  https://v4musclebike.com/threads/starter-relay-question.27094/  
  Полезно для обсуждения номиналов предохранителя и рисков с aftermarket relay.

- Starter fuse blows when battery is connected V45 Sabre  
  https://v4musclebike.com/threads/starter-fuse-blows-when-battery-is-connected-v45-sabre.46738/  
  Полезно тем, что ясно объясняет, почему неправильно распинованные aftermarket solenoids могут создать прямое КЗ.

- VF700s Starter Selenoid Replacement  
  https://v4musclebike.com/threads/vf700s-starter-selenoid-replacement.46535/  
  Полезно для проверки расположения проводов на replacement solenoids для VF700S.

- Rebuild wiring harness?  
  https://v4musclebike.com/threads/rebuild-wiring-harness.35483/  
  Полезно, если жгут нужно не ремонтировать локально, а действительно перебрать заново.
