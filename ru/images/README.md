# Референсы по изображениям

Эта папка собирает визуальные материалы для проекта `1982 Honda VF750S Sabre`.

## Папки

- `stock/` — фотографии стокового Sabre
- `cafe-racer/` — уже собранные фото готовых кастомов
- `pinterest/` — визуальное вдохновение из Pinterest
- `build-process/` — фотографии процесса сборки и конверсии из интернета
- `repair-service/` — миниатюры сервисных и ремонтных референсов из интернета
